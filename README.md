homebrain-stt-client
====================

#### Dependencies

- swig

Installable from PyPi with pip

- pocketsphinx
- pyalsaaudio
- espeak (optional)

#### Commands

Lights/Lamps: Toggles homebrain lamps
quit: exits the stt program
abort: aborts command
